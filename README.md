# Proyecto Final - TechLab API

API REST desarrollada con Node.js y Express para la gestión de productos, utilizando Firestore como base de datos en la nube.

## Requisitos
- Node.js v18+
- Cuenta en Firebase

## Instalación
1. Clonar el repositorio.
2. Ejecutar `npm install`.
3. Configurar las variables de entorno en un archivo `.env` basado en la configuración de Firebase.
4. Iniciar con `npm run start`.
